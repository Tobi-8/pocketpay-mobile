# Contributing to PocketPay Mobile

Thank you for your interest in contributing to PocketPay Mobile! We welcome pull requests, bug reports, and feature requests.

## Getting Started

1. Fork the repository and clone it locally.
2. Install dependencies using `npm install --legacy-peer-deps`.
3. Follow the setup instructions in the [README.md](README.md) to start the Expo development server.

## Accessibility

We strive to build a wallet that is accessible to everyone. Before submitting a pull request for any UI changes, please review and verify your work against our [Accessibility Checklist](docs/accessibility.md). 

## Submitting a Pull Request

- Create a new branch for your feature or bugfix.
- Follow our coding standards and ensure your code builds without errors.
- Write clear, descriptive commit messages.
- Reference any relevant issues in your pull request description.
